## Описание проекта course_work_3

### Проект содержит файлы:
+ data/operations.json - файл json с которым происходит работа программы
+ utils/func.py - файл содержащий функции для работы с json файлом
+ main.py - основной файл вызывающий функции из func.py

### Описание функций из файла func.py:
 + load_json_file - загрузка json файла, принимает путь до json 
 + get_filter_json_file - фильтрация json файла, принимает загруженный json
 + get_convert_check - формирование номера карты или счета со скрытием цифр
 + get_convert_date - формирование необходимого формата даты
 + get_info_transaction - получение готовой информации о транзакции
 + get_select_info_transaction - получение списка транзакций успешных или безуспешных и вывод пользователю
